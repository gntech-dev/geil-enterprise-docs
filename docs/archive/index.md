## Use rules

- Archive documents are not active implementation guidance.
- Active GEIL implementation must follow the current Foundation, Platform, Microsoft Core, Security, Architecture, and Operations sections.
- Archive documents may explain why a previous direction changed, but they must not override accepted ADRs or pilot-validated implementation guides.
